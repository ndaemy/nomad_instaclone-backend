# nomad_instaclone-backend

Instagram clone with Express + Prisam

- [x] Create account
- [x] Request secret
- [x] Confirm secret (Signin)
- [x] Like / Unlike a post
- [x] Comment on a post
- [x] Search by user
- [x] Search by location
- [x] Follow user
- [x] Unfollow user
- [x] Edit my profile
- [x] See user profile
- [x] See my profile
- [x] See the full post
- [ ] Upload a photo
- [ ] Edit / Delete the photo
- [ ] See the feed
